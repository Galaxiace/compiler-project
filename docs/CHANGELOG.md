## [1.0.0] — Sprint 8 (Final)

### Added
- Полный CLI (`mycc`) с Unix-совместимым интерфейсом
- Режимы: `-S` (assembly), `-c` (object), `-E` (preprocess), `-O0-3` (optimization levels)
- `--ast`, `--ir`, `--stats`, `--verbose`
- `-Wall`, `-Werror`, `--format=json`, `--color`, `--max-errors`
- Единая система ошибок с кодами (E001-E599, W001-W299)
- Контекстные подсказки и позиционирование в сообщениях об ошибках
- 316 тестов, 80% покрытие кода
- Makefile для сборки и установки
- Полная документация (README, language_spec, grammar, developer, CHANGELOG)

### Changed
- Новый `mycc.py` как точка входа (вместо `lexer/cli.py`)
- Улучшенные сообщения об ошибках (позиция, контекст, подсказки)
- Оптимизирован `ir/optimizer.py` (96% reduction на demo)

## [0.7.0] — Sprint 7 (Расширенные возможности и оптимизации)

### Added
- Поддержка массивов: объявление, инициализация, доступ по индексу
- Поддержка структур: объявление, доступ к полям, возврат из функций
- Параметры-массивы в функциях
- Constant Folding (свёртка констант)
- Constant Propagation (распространение констант)
- Dead Code Elimination (удаление мёртвого кода)
- Unreachable Code Elimination (удаление недостижимых блоков)
- Jump Optimization (оптимизация переходов)
- Пайплайн оптимизаций с итерациями до стабилизации
- Статистика оптимизаций (`--stats`)
- Демо-программы: quicksort, optimization_demo

## [0.6.0] — Sprint 6 (Управляющие конструкции и короткое замыкание)

### Added
- Короткозамкнутые логические операторы `&&` и `||`
- Полная поддержка if/else с вложенными условиями
- Поддержка while с обратным ребром в CFG
- Поддержка for (раскрывается в while-эквивалент)
- Float-сравнения через `ucomiss` с обработкой NaN
- Реляционные операторы: `<`, `<=`, `>`, `>=`, `==`, `!=`
- Поддержка NOT (`!`) оператора

## [0.5.0] — Sprint 5 (x86-64 Кодогенерация)

### Added
- x86-64 кодогенератор (NASM синтаксис)
- System V AMD64 ABI (параметры в rdi/rsi/rdx/rcx/r8/r9, xmm0-xmm7)
- Управление стековым фреймом (пролог/эпилог, выравнивание 16 байт)
- Полный маппинг IR-инструкций на x86-64
- Поддержка float (SSE инструкции: movss, addss, subss, mulss, divss)
- Секции данных: `.text`, `.data`, `.bss`, `.rodata`
- Runtime библиотека: `print_int`, `print_string`, `read_int`, `exit`, `_start`
- Статическая линковка исполняемых файлов
